---
title: ## System Requirement
---
### server
1. CPU  : Minimal Intel 4-core .
2. RAM  : Minimal 16 GB .
3. Storage  : Minimal 100 GB SSD untuk data dan file.
4. Network  : Internet minimal 100 Mbps.

### pengguna
- desktop 
1. CPU  : Minimal Intel i3 atau setara
2. RAM  : Minimal 4 GB
3. Browser  : Chrome, Firefox, Edge versi terbaru.

- mobile
1. OS       : Android 10 / iOS 14 atau lebih baru
2. RAM      : Minimal 2 GB
3. Storage  : Minimal 500 MB ruang kosong untuk aplikasi dan data
4. Browser  : Chrome Mobile, Safari Mobile

## Kebutuhan Perangkat Lunak
- Server
1. Web Server           : Apache.
2. Database             : MySQL atau PostgreSQL.
3. Bahasa Pemrograman   : PHP dan JavaScript.
4. Framework            : Bootstrap.
5. OS                   : Linux (Ubuntu, CentOS, atau Debian).

- Pengguna
1. Web Browser          : Versi terbaru dari  Chrome, Firefox, Edge, dan Safari (terbaru).
2. Responsif            : Desain harus bekerja optimal pada layar dengan resolusi mulai dari 360x640 (mobile) hingga 1920x1080 (desktop).

## Kebutuhan Jaringan
1. HTTP/HTTPS       : Untuk komunikasi aman.
2. Firewall         : Lindungi server dari akses yang tidak sah.
3. Sertifikat SSL   : Untuk enkripsi data.

## Kebutuhan Keamanan
1. Enkripsi Data        : Gunakan HTTPS.
2. Perlindungan DDoS    : Lindungi dari serangan.
3. Sistem Autentikasi   : Pastikan akun pengguna aman.
4. Backup Otomatis      : Simpan data penting secara otomatis.
5. Rencana Pemulihan    : Siap untuk bencana.

## 3. Kebutuhan Fungsional
1. Sistem dapat menangani hingga 10.000 pengguna secara bersamaan.
2. Fitur pencarian produk dengan filter kategori, harga, dan merek.
3. Integrasi pembayaran online (midtrans, PayPal, dll.).
4. Sistem login dengan autentikasi 2 langkah (OTP melalui email atau SMS).
5. Dashboard admin untuk manajemen produk, kategori, dan manajemen pengguna.

## Kebutuhan Non-Fungsional
- Kinerja
1. Waktu Muat           : Halaman harus muat dalam kurang dari 3 detik.
2. Pengguna Bersamaan   : Bisa tangani minimal 1000 pengguna sekaligus.
3. Pencarian Cepat      : Hasil pencarian produk dalam kurang dari 2 detik.

- Usability
1. Antarmuka        : Harus mudah dipakai dan navigasi jelas.
2. Responsif        : Bisa diakses di desktop, tablet, dan smartphone.

- Kompatibilitas
1. Browser          : Harus bisa diakses di berbagai browser modern.
2. Sistem Operasi   : Bisa diakses di Windows, macOS, Android, dan iOS.
