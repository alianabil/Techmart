
## Analisis Kebutuhan Fitur dan Fungsi
   - Fitur Utama :
     - Pencarian Produk: Pengguna dapat mencari produk berdasarkan kategori, harga, dan merek.
     - Keranjang Belanja: Memungkinkan pengguna untuk menambahkan produk, melihat subtotal, dan melanjutkan ke checkout.
     - Pembayaran Aman: Berbagai metode pembayaran, termasuk e-wallet, kartu kredit, dan transfer bank.
     - Pelacakan Pesanan: Pengguna dapat melacak status pengiriman produk.
     - Promosi dan Diskon: Pengguna dapat menggunakan kupon atau mengikuti promosi khusus.
     - Ulasan Produk: Pengguna dapat memberikan dan melihat ulasan produk yang dibeli.

   - Fitur Admin :
     - Manajemen Produk: Admin dapat menambah, mengubah, atau menghapus produk.
     - Manajemen Transaksi: Admin dapat memproses dan memantau pesanan yang dilakukan.
     - Laporan Penjualan dan Stok: Admin dapat melihat laporan penjualan dan memantau stok barang.
